# Analizador AM2

Herramienta de visualización y análisis de superficies para Análisis Matemático II. 

## Características
- Visualización 3D de cuádricas y funciones personalizadas.
- Cálculo de derivadas parciales, matriz Hessiana y plano tangente.
- Estudio de límites por trayectorias y coordenadas polares.
- Dashboard integral de reporte técnico.

## Cómo ejecutar localmente
1. Clonar el repo.
2. Instalar dependencias: `pip install -r requirements.txt`
3. Ejecutar: `streamlit run src/app.py`
